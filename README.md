Number Guessing Game
A simple command-line game where the user tries to guess a number between 1 and 100.

How to Run
1. Make sure you have Python 3 installed.
2. Navigate to the number-guessing-game directory.
3. Run the script from your terminal:

python game.py
How to Play
1. The program will think of a random number.
2. Enter your guess when prompted.
3. The program will tell you if your guess is too high or too low.
4. Keep guessing until you find the number!
